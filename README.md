関連Issue：<https://github.com/VOICEVOX/voicevox_core/issues/1001>

## 動かす

```bash
pnpm run dev
```

## フォーマット

```bash
pnpm run lint
pnpm run fmt
```

このリポジトリでは、Markdownファイルに対して特殊なフォーマットを適用しています。
詳細は`./tools/format.ts`を参照してください。
